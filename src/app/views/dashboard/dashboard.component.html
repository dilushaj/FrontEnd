Data Preprocessing
Data Pre-processing consists of a data cleaning process in which issues in datasets that make it unusable are handled. Duplicate molecular sequences, incomplete datasets and noisy values can be listed as some of these issues. Data pre-processing is a vital part of effective phylogenetic analysis since it greatly affets the accuracy of phylogenetic inference utilizing a molecular dataset.
Phylogenetic Inference
Phylogenetic Inference is the process of generating a phylogenetic tree, which is a structure that represents the evolutionary relationships among a group of taxological units. This process can be accomplished in different ways. Maximum Likelihood, Bayesian Approach, Maximum Parsimony, Neighbor-Joining method and UPGMA method are the most prominant algorithms. The effectiveness of each algorithm depends on dataset features.
Tree Visualization
To review the results of phylogenetic inference, the visualization of the generated tree is essential. Many visualization layouts and techniques exist, such as cladograms/phylograms and rectangular/circular/radial, with different strengths and weaknesses. A visualization layout should be selected depending on the requirement of the phylogenetic analysis or the features of the dataset.
